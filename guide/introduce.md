set-components 内部集成了antd的一些组件，和日常开发的封装 inspire by antd-doddle

## 安装

set-components是一个多包的项目，组件与方法分离。

安装包
```sh
npm i set-components --save
```

## 使用  

使用包中的组件  
```javascript
import { Table } from 'set-components';
```

使用包中的方法 
```javascript
import { getEnumObject } from 'set-components/utils';
```
