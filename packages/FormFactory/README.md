## API

### DynamicForm 输入
| 参数名 | 作用 | 必传 | 类型 | 默认值  
:--|:--:|---:|---:|:---:
| formBasicConfig | 排列属性， 其中包括三个属性（grid，formItemWidth，formName） | 是 | object | -- 
| renderForm | 表单内容 | 是 | array | 
| btnList | 按钮列表 | 否 |  array | 
| ref | 表单引用 | 是 |  string |
| ... | table其他可设置属性 | 否 | -- | --
